You can download below the "addresses-only" and full version of the Open Addresses database.

The data is updated every day.

You will need a bittorrent client in order to download the data. We recommend:

  • Windows: µTorrent
  • OSX / Linux: Transmission

Open Addresses Database (downloadable, addresses-only)

This is the addresses-only version of the Open Addresses database, which just includes the complete list of addresses available at the moment of publishing, without provenance information. It is distributed as a single ZIP file, containing one ZIP file per postcode initial letter, containing one CSV file per postcode area.

  • Format: zip
