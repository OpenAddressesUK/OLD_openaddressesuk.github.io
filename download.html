Open Addresses Database Download

The Open Addresses database is available both live through our publishing APIs and for download on this page.

The data was generated from the address sources in our live database on 10 December 2014.

We plan to publish updated versions for download every time one of the current sources of address data changes. As soon as we will start processing end-user contributions in Beta, the data for download will be re-generated nightly.

The Open Addresses dataset is licensed as Open Data under the Creative Commons Attribution 4.0 International (CC BY 4.0) licence . The database contains data that, in its original form, was previously licensed under the UK Open Government Licence or the Ordnance Survey Open Data Licence, subject to Crown copyright and Crown database rights. Read more about our sources here.

Published by Open Addresses Ltd.

You will need a BitTorrent client in order to download the data. We recommend:

  • Windows: µTorrent
  • OSX / Linux: Transmission

Open Addresses Database (downloadable, addresses-only)

This is the addresses-only version of the Open Addresses database, which just includes the complete list of addresses available at the moment of publishing, without provenance information. It is distributed as a single ZIP file, containing one ZIP file per postcode initial, containing one CSV file per postcode area. Each CSV line is one address.

  • Format: zip (BitTorrent)

Open Addresses Database (downloadable, full)

As above, but including provenance information. The CSV files do not have one line per address, but one line per provenance step being described. The actual address is repeated every time.

  • Format: zip (BitTorrent)
